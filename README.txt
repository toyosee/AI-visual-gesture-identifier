Open index file directly in browser or using live server in VS Code.
- Allow webcam to be started by application
- Show a finger gesture from 1 fingers to 5 fingers directly to the camera. Keep fingers infront of your face.
- Results will be generated automatically, based on how stable your hand is and how clear camera is.
- The tool tries to capture your finger gesture and identifies index finger as 1.
- This tool is trained with limited data and still being trained

- This tool captures and interprts your gesture and model predicts what it is. when you click on the identify button, final prediction is printed out on screen

- This model is not 100% accurate. it is still in training phase.
